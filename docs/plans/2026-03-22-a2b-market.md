# A2B Market Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a demo `a2b market` capability that syncs a GitHub-backed market index, caches it locally, and lets AI browse categories and entries through CLI commands.

**Architecture:** Keep market content in the separate `a2b-market` GitHub repository, then add a thin local cache and read-only CLI browser in the main A2B repo. The first version only supports updating, listing, showing, and searching entries; it does not execute recipes or auto-install external tools.

**Tech Stack:** Node.js CLI, local JSON/Markdown cache, GitHub raw content fetches, Vitest for tests
